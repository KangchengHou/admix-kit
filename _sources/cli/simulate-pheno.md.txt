# Phenotype simulation

```bash
admix simulate-admix-pheno \
    --pfile <plink file prefix> \
    --hsq <heritability> \
    --cor <correlation> \
    --n-causal <number of causal SNPs> \
    --out-prefix <output prefix>
```
